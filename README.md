# RoomBookerApp

* ----components/
* -------App.js
* -------CurrentTimeAndEvents.js
* -------EventInProgress.js
* -------EventItem.js
* -------EventItemInProgress.js
* -------EventList.js
* ----images/
* -------event-icon-color.png
* -------event.jpg
* ----ajax.json
* ----timeUtil.js
*-index.js
